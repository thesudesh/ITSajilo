Eighth Semester

Game Technology

Course Title: Game Technology

Course No.: CSC465

Course Nature: Theory

Semester: Eighth

Year: Fourth

Full Marks: 60 + 20 + 20

Pass Marks: 24 + 8 + 8

Credit Hours: 3

Course Description: This course is a practical and conceptual introduction to game design and development including basic ideas of game design, learn to design a game, and working as a game designer. This course will provide ample opportunities to try out concepts and theories to design, develop and test 2D and 3D games. The main platform will be Unity, a cross-platform game editor and engine widely in use by many companies in the game industry.

Course Objectives: After completion of the course, students will learn basics of game development, to design games, to design games, to use Unity game editor and engine to develop games.
